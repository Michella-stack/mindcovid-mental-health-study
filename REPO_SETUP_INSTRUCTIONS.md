# GitHub Repo Setup Instructions
### Research: Mental Health Impacts on Healthcare Workers During COVID-19

---

## Step 1 — Create a new GitHub repository

1. Go to [github.com/Michella-stack](https://github.com/Michella-stack)
2. Click the green **"New"** button (top left)
3. Name it: `mindcovid-mental-health-study`
4. Set visibility to **Public** (so it appears on your portfolio)
5. Check **"Add a README file"** → we will replace this with our version
6. Click **"Create repository"**

---

## Step 2 — Upload your files

Upload these files to the repo:

| Upload this | Do NOT upload this |
|---|---|
| ✅ `demochangesfinal.pdf` (your essay) | ❌ `bd_Warren_18122020.xlsx` (the raw data — restricted by your IMIM agreement) |
| ✅ `Formulario_MINDCOVID.pdf` (data request form) | |
| ✅ `README.md` (provided — see below) | |

To upload: go to your repo → **"Add file"** → **"Upload files"** → drag and drop → **Commit changes**.

---

## Step 3 — Replace the README

After creating the repo, GitHub will have an empty README. Replace it:

1. Click the pencil ✏️ icon on the README.md file
2. Delete everything inside
3. Paste the full contents of the `README.md` file provided to you
4. Click **"Commit changes"**

---

## Where exactly does the IMIM credit go?

The data agreement (Condition V) requires this exact text to appear **in any publication of results**. Here is where to place it:

### In the README.md ✅
Already included in the section titled **"Data Attribution / Atribución de los datos"**:

> *Este estudio se ha realizado utilizando los datos anonimizados cedidos por la Fundació Institut Hospital del Mar d'Investigacions Mèdiques en el marco del Estudio Mind Covid ISCIII FEDER COV20/00711.*

### In your essay PDF ✅
The credit should appear in your essay. Check your current PDF — it references MINDCOVID throughout but **does not include the exact required formula from Condition V**.

You have two options:
- **Option A (simplest):** Add a note to the **last page or acknowledgements section** of your essay PDF before uploading. Open the PDF in Word or Google Docs, add a line at the bottom of the bibliography page:
  > *Datos: Este estudio se ha realizado utilizando los datos anonimizados cedidos por la Fundació Institut Hospital del Mar d'Investigacions Mèdiques en el marco del Estudio Mind Covid ISCIII FEDER COV20/00711.*
- **Option B:** Leave the essay PDF as-is (it was submitted to UPF in its current form) and rely on the README credit. This is likely sufficient for a portfolio context.

### On your portfolio website ✅
Already added to `research.html` — the small italic line under the MINDCOVID card reads the required attribution in Spanish.

## Summary of what is and is not allowed

| Action | Allowed? | Why |
|---|---|---|
| Share your essay publicly on GitHub | ✅ Yes | It's your original written work |
| Share the data request form PDF | ✅ Yes | It's a document about your study |
| Publish charts / graphs from your analysis | ✅ Yes | These are your derived outputs |
| Upload the raw `.xlsx` dataset | ❌ No | Condition IV of the IMIM agreement prohibits this |
| Add this to your portfolio website | ✅ Yes | With the required IMIM credit attribution |
| Reference the study in a CV or LinkedIn | ✅ Yes | Standard academic practice |
